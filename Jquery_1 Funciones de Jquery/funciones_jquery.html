I. Efectos: Para crear elementos de animación

El método .hide () ocultará la información, pero por el contario, el método .show() lo mostrará.

El método .toggle() alterna entre .hide() y .show() para los elementos seleccionados sin necesidad de duplicar líneas de código.

El método .slideUp() ocultará la información deslizándose hacia arriba, por el contario, el método .slideDown() la deslizará hacia abajo. Puedes elegir que se deslice "fast" (rápido) o bien "slow" (lento).

El método .slideToggle() alterna entre .slideUp() y .slideDown() para los elementos seleccionados sin necesidad de duplicar líneas de código. Puedes elegir que se deslice "fast" (rápido) o bien "slow" (lento).

El método .fadeOut() ocultará un elemento en forma gradual, por el contario, el método .fadeIn() lo mostrará.
Este método tiene dos páramentros, uno es la velocidad (en milisegundos o en valores fijos como "fast" -rápido- o bien, "slow" -lento-).
Y el otro parámetro es el call back, que le da la opción de insertar otra función a ejecutar una vez que el desvanecimiento se ha completado.

Manipulación: Recuperar o establecer un valor o texto en cualquier elemento HTML

El método .before() inserta el contenido especificado delante (antes) de los elementos seleccionados. El método .after() hará exactamente lo contrario.

El método .append() inserta el contenido especificado al final de los elementos seleccionados.

En un documento HTML, se puede usar .html() para obtener el contenido de cualquier elemento. Si la expresión del selector coincide con más de un elemento, solo se devolverá el contenido HTML de la primera coincidencia.
